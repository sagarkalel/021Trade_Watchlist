# 021 Trade - Watchlist

## What I Built

I replicated the Watchlist screen as closely as possible to the provided video. It includes a top stocks header showing SENSEX and NIFTY BANK, a horizontal tab bar to switch between watchlists, and a stock list showing price, change, and percentage for each stock. From the Edit screen, you can drag to reorder stocks, delete a stock, and rename the watchlist. All data is mocked locally in a repository class to keep things simple.

## Scope

The focus of this assignment was the Watchlist screen only. The bottom navigation bar (Orders, GTT+, Portfolio, Funds, Profile) is present for UI completeness but those screens are not implemented. Similarly, the Sort by button and search bar are placeholders — only the core requirement of reordering stocks has been fully implemented.
